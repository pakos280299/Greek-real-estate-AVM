# Greek Real-Estate AVM Prototype

Ongoing personal project: Hedonic pricing model on ~17k Greek listings from Kaggle.

- **Data:** Public Greece Property Listings (2022 sample).
- **Tools:** Python (pandas, scikit-learn, GeoPandas), R (data.table, leaflet), SQL (SQLite).
- **Key Features:** Data cleaning, geospatial engineering (dist to metro), Random Forest model (R² ~0.65+), interactive leaflet map.
- **Demo:** Open avm_notebook.ipynb in Jupyter or view map in avm_map.html.

Run in Kaggle: [https://www.kaggle.com/code/korniotisfilippos/avm-project].
